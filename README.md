# Brain Tumor Detection using CNN and MobileNetV2

## Overview
This project detects brain tumors from MRI images using Deep Learning and Transfer Learning techniques with MobileNetV2 architecture.

## Features
- MRI Image Classification
- Binary Classification
- Deep Learning Prediction
- Transfer Learning using MobileNetV2
- TensorFlow and Keras based implementation

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- MobileNetV2

## Project Files
- train.py → Model training
- predict.py → Prediction script
- app.py → Main application
- split_dataset.py → Dataset splitting
- merge.py → Dataset merging
- gpu.py → GPU configuration

## Dataset
Dataset was too large to upload on GitHub.

Dataset Link:
https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection

## How to Run

```bash
pip install -r requirements.txt
python train.py
```

## Model
The project uses CNN with MobileNetV2 for feature extraction and classification.
